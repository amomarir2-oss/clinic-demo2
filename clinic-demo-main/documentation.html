Site Documentation

A brief guide to the structure and editable parts of this demo site.

Getting started

Open style.html to edit the main homepage layout. Styles are in styles.css, and client-side JS behavior is in Scripts.js.

Components

  • Navbar — markup in style.html near the top; look for .navbar.
  • Hero — the hero uses .hero-section and is powered by code in Scripts.js.
  • Help selector — the patient/help selector is in style.html and its behavior is in Scripts.js. The help options grid was intentionally removed from the homepage.

Editing guidelines

When updating layout or styles, keep changes small and test in a local browser. Use semantic HTML and avoid heavy inline styles where possible.
